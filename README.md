# iso-location-code
Parsing ISO Lcoation code from MDB-file and export data into JSON-file

## Usage: 
```
java -jar mdbgrabber.jar
```

## Help: MDBGrabber
```
 -?                  Show help
 -h,--help           Show help
 -i,--input <arg>    Path until *.MDB database
 -o,--output <arg>   Path until output JSON file, if not getting export in
                     'out.json'-file
 -s,--stat           Show stat
 -v,--version        Show version
```
